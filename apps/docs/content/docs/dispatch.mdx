---
title: Dispatch a call
description: Create a typed, paid call request.
---

`POST https://agentcaller.io/api/v1/calls`

Supply `Authorization: Bearer ac_…` and a unique `Idempotency-Key` header. The first request returns an x402 challenge. Retry the exact request with `PAYMENT-SIGNATURE`.

```json
{
  "destination": "+34915200000",
  "destinationCountry": "ES",
  "language": "es",
  "maxDurationSeconds": 600,
  "maxAmountUsd": 1.2,
  "task": {
    "type": "reservation",
    "partySize": 2,
    "timeWindow": {
      "startsAt": "2026-07-14T19:00:00.000Z",
      "endsAt": "2026-07-14T21:00:00.000Z",
      "timezone": "Europe/Madrid"
    },
    "contact": { "name": "Marta Ruiz" }
  }
}
```

## Limits

`maxDurationSeconds` is enforced on the call itself: AgentCaller hangs up when it elapses.

`maxAmountUsd` is a billing ceiling, not a call timer. It must cover the destination's connection
fee and one started minute, and you are never charged more than it — but it does not by itself end
a call. Size `maxDurationSeconds` to the spend you are willing to authorize.

You are charged the connection fee plus every started minute, capped at `maxAmountUsd`. A call that
connects and fails still incurs the connection fee and one started minute.

## Voices

`voiceId` is optional and defaults to the platform voice for your language. Supplying one that is
not enabled for your client returns `422`. Ask an operator to enable a voice before using it.

## Correlation

`clientReference` is an optional free-text field, echoed back on the call resource and in list
responses. Use it to tie a call to your own records.
